Clearfix. When an element only contains floated elements, it collapses on itself.
To prevent this behavior, you have to “clearfix” it.

Long strings truncation. Use .break for preventing the text to break out of their container (such as an URL) by forcing the break.
And .ellipsis to make an ellipsis in case the string is too long for its container.
Note: set the width of the .ellipsis element to suit your needs.
