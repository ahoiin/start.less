Mixins for LESS Designers

Color for the placeholder text. For input elements.

.placeholder(@color);

Create a new webfont. Requires the name of the webfont and url to the .eot, .woff and .ttf files.
Note: the mixin must be in the font-face tag. 

@font-face { .createFontFace(@fontName, @fontPath); }

Round Corners.

.borderRadius(@radius); //e.g. 2px

Opacitiy in full per cent

.opacity(@opacity); //e.g. 50

Placement. @type could be absolute, relative etc.

.placement(@type, @top, @right, @bottom, @left);

Clipping currently supports only rect and only works with position: absolute; So just enter the values for the rect.
Note: Both the top and the bottom values define the offset from the top border and the left and right values define the offset from the left border.

.clip(@top, @right, @bottom, @left);

CSS3 Transitions

.transition(@transition); //e.g. all 0.3s ease-out
.rotate (@degrees); //e.g. 90deg
.translate(@x, @y); //e.g. 100px, 100px

CSS3 Animation

.animation(@val) //e.g. spin .6s infinite linear 
.keyframeAnimation (@name, @from, @to); //e.g. spin, transform:rotate(0deg), transform:rotate(360deg)
