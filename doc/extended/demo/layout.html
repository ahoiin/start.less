Responsive Layout

As the default, the layout adapts itself to the screensize and the following items: header, nav section, section, article, footer section, .container get a specific width. However, if you want to use the full width, just use the class .full-width.

Centered Layout

Due to the automatic width of all important elements, the whole site can easily be centered by using the class .center. The class can also combined with the grid system.

...

The navbar and footer should be include an element which has the class .center so the whole footer and navbar can use the whole width and only the content is centered.
